Reviewer's Note

Selamat! Anda telah berhasil menyelesaikan tugas untuk Submission 1 - Aplikasi Github User.

Notes:

  • Jangan lupa untuk memeriksa hasil review kode yang diberikan pada table Submitted App.
  • Pada sebuah ImageView, jangan lupa untuk tambahkan attribut android:contentDescription. Ini mendefinisikan konten dengan teks yang menjelaskan secara singkat konten tampilan. Properti ini digunakan terutama untuk aksesibilitas. Karena beberapa tampilan tidak memiliki representasi tekstual, atribut ini dapat digunakan untuk menyediakannya.
  • Selalu perhatikan resources yang tidak pernah digunakan di dalam project karena akan mempengaruhi size APK nantinya. Anda bisa memanfaatkan fitur Remove Unused Resource untuk menghapus resources yang tidak pernah digunakan.
  • Dalam menulis sebuah kode di dalam sebuah kelas, selalu perhatikan import yang tidak pernah digunakan agar kode yang sudah ditulis menjadi lebih bersih dan enak dilihat. Anda bisa melakukan reformat code dan optimized import supaya code lebih rapi dan bersih dari import yang tidak digunakan.
  • Sebaiknya pada beberapa cuplikan ini, gunakanlah warna yang kontras, seperti untuk warna teks harus lebih menonjol daripada backgroundnya, agar lebih terlihat teksnya ya :)



Tetap semangat menyelesaikan kelas Belajar Fundamental Aplikasi Android.
